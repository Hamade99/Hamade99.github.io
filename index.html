Exceptionally Uncertain Model Mining

2023-2024
In Progress

The challenge is to obtain exceptional subgroups in a dataset. We developed a novel method, EUMM, that bridges the gap between subgroup classification and Uncertainty Quantification (UQ).

The novel model was applied to eight regression datasets, where the meaning of each subgroup is further investigated. As an uncertainty quantification method, Deep Ensembles were used.

Research awarded with a 10 in the corresponding class. The paper is currently in the process of being published in IDA 2024.

Technologies Used

Python TensorFlow Uncertainty in AI Model Selection Machine Learning Model Representation Deep Ensembles

Automated Sarcasm Detection in Text

January 2022
Completed

Bachelor thesis concerning the thorough analysis of English and Arabic text to discern sarcastic from non-sarcastic tweets.

Performed feature extraction on large datasets using state of the art NLP methods using BERT for both English and Arabic. Developed several optimized machine learning models through AutoML.

Final result shows the differences in features that define sarcasm in both languages, as well as a high accuracy to classify text as sarcastic or non-sarcastic.

Technologies Used

NLP Machine Learning Feature Extraction Data Mining Text Analysis BERT Python AutoML
